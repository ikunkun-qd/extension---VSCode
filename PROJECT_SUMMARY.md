## 🚀 部署和使用

### 安装方式
1. **VSIX 文件安装**（推荐）
   ```bash
   code --install-extension component-doc-helper-1.0.0.vsix
   ```

2. **从源码构建**
   ```bash
   npm install
   npm run compile
   npm run package
   ```

### 快速配置
1. 点击 VSCode 左侧"组件文档"图标
2. 选择 "OurYun" 模板
3. 设置文档路径
4. 保存配置

### 使用方法
1. 打开包含组件的文件
2. 将鼠标悬浮在组件名上
3. 查看悬浮提示
4. 点击链接查看完整文档
